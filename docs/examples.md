# Examples

The primary example walkthrough now lives in [`../examples/README.md`](../examples/README.md).

Use it when you want:

- the recommended order to read the examples
- the supported `npm run example:*` commands
- the build prerequisite for direct `npm exec tsx examples/...` runs
- the numbered onboarding set
- the list of advanced examples

The top-level examples are intentionally small and self-contained. Advanced integrations live under `examples/advanced/`.
